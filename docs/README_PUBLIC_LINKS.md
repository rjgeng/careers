## Explicitly Not Public

The following are intentionally **not** linked publicly:

These exist to support internal clarity only.

## Promotion Rule

A document may be promoted to public status only if it is:

- Non-proprietary
- Emotion-neutral
- Interview-safe
- Useful without additional explanation
