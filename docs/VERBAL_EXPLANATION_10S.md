# 10-Second Verbal Explanation (Ultra-Short)

Use this when time is tight or early in a recruiter screen.

---

## 10-Second Version

> I treat my career like a system — ATS-safe resumes, interview prep, and role selection — so the process stays focused and predictable.

---

## When to Use

- Recruiter cold calls
- First 30 seconds of a screen
- Quick introductions

---

## Rule

Say it once, then pause.

If they’re interested, they’ll ask.
